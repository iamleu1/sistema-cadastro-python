# Sistema de Cadastro em Python

Este é o meu **primeiro projeto em Python**, desenvolvido com o objetivo de praticar conceitos básicos de programação e organização de código.

# Sobre o projeto

O sistema funciona no terminal e permite:

* 📄 Visualizar pessoas cadastradas
* ➕ Cadastrar novas pessoas
* 💾 Salvar os dados em um arquivo `.txt`

Os dados são armazenados de forma simples em um arquivo local (`pessoas.txt`), simulando um pequeno banco de dados.

# Tecnologias utilizadas

* Python
* Manipulação de arquivos
* Estruturas de repetição (`while`)
* Condicionais (`if/elif`)
* Modularização com funções



Este projeto foi desenvolvido como parte do meu aprendizado em programação, com foco em:

* Lógica de programação
* Organização de código
* Interação com o usuário no terminal
